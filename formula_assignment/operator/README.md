# C の演算子
C には、次の単項演算子が用意されています。
```
シンボル	名前
- ~ !	    否定演算子と補数演算子
* &	      間接演算子とアドレス演算子
_Alignof	アラインメント演算子 (C11 以降)
sizeof	  サイズ演算子
+	        単項プラス演算子
++ --	    単項インクリメント演算子と単項デクリメント演算子
```

二項演算子は、左から右へ結合されます。 C には、次の二項演算子が用意されています。
```
シンボル	名前
* / %	          乗算演算子
+ -	            加法演算子
<< >>	          シフト演算子
< > <= >= == !=	関係演算子
& | ^	          ビットごとの演算子
&& ||	          論理演算子
,	              順次評価演算子
```
